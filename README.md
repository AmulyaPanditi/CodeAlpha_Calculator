# Modern Calculator Web Application

A responsive and interactive calculator web application developed using HTML, CSS and JavaScript.

## Features

- Responsive Design
- Modern Glassmorphism UI
- Keyboard Support
- Real-Time Calculations
- Smooth Hover Animations
- Clean User Experience

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Functionalities

- Addition
- Subtraction
- Multiplication
- Division
- Percentage Calculations
- Delete Function
- Clear Screen
- Keyboard Interaction Support

## Project Preview

This project demonstrates frontend development concepts including:
- DOM Manipulation
- Event Handling
- Responsive Design
- Interactive UI Development
